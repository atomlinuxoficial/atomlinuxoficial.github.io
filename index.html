Sobre

O Atom Linux busca entregar uma experiência amigável para iniciantes sem limitar usuários avançados. O objetivo é oferecer estabilidade, desempenho e compatibilidade para trabalho, estudo e jogos.

Base do Sistema

Debian Stable

Base sólida do sistema. Prioriza estabilidade, confiabilidade e suporte de longo prazo.

KDE Plasma

Interface moderna, elegante e altamente personalizável para todos os tipos de usuários.

APT

Gerenciador oficial de pacotes para instalar e atualizar programas com segurança.

Flatpak

Instale aplicativos universais e mantenha versões recentes dos programas.

AppImage

Aplicativos portáteis que funcionam sem necessidade de instalação.

Edições

Amigável

Voltada para iniciantes com configuração simplificada, atualizações automáticas e instalação fácil de programas.

Pro

Voltada para usuários avançados que desejam controle total e máxima personalização.

Jogos

Steam

Milhares de jogos disponíveis com suporte ao Proton.

Wine

Execute programas e jogos do Windows diretamente no Linux.

Lutris

Gerencie bibliotecas de jogos de diferentes plataformas.

Proton GE

Compatibilidade aprimorada para títulos modernos e antigos.

Segurança

Firewall Ativo

Proteção de rede habilitada por padrão.

AppArmor

Restringe permissões de aplicativos para maior segurança.

Atualizações Automáticas

Correções críticas instaladas rapidamente.

Repositórios Assinados

Verificação digital dos pacotes instalados.
