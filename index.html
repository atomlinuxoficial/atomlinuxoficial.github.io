Sobre

Atom Linux é uma distribuição baseada em Debian que busca equilíbrio entre estabilidade, compatibilidade com jogos Windows e liberdade de personalização.

Filosofia

"Comece fácil. Cresça sem limites."

O sistema foi projetado para evoluir com o usuário, sem obrigar migração para outra distribuição.
